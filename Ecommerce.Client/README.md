# Ecommerce Client (Angular 20)

The frontend component of the Ecommerce application is a Single Page Application (SPA) designed to provide a premium, fast, and responsive user experience. It is built using the latest **Angular 20** framework.

## 🎨 UI & Design Ecosystem

The application leverages multiple premium UI libraries to fetch a rich and stunning user interface, complete with responsive layouts, dark mode support in dashboards, and custom animations.

*   **PrimeNG & PrimeFlex:** Core UI component library used for powerful data representation (tables, inputs) and layout structuring using PrimeFlex utilities.
*   **Bootstrap & ng-bootstrap:** Used for foundational grid layouts, modals, and responsive components.
*   **DataTables:** Advanced tabular data handling inside the Admin dashboard components.
*   **Chart.js & ng2-charts:** For rendering analytical charts and metrics in the dashboard.
*   **FontAwesome & PrimeIcons:** Comprehensive iconography suites.
*   **SweetAlert2 & ngx-toastr:** Provides elegant, non-intrusive notifications and confirmation dialogues.

## 🛠️ Architecture & Patterns

*   **Component-Based Architecture:** Features isolated, reusable, and heavily encapsulated Angular components.
*   **Modular Design:** Features are split into logical modules (e.g., Auth, Dashboard, Products, Checkout) allowing for clear boundaries and lazy-loading routes.
*   **Reactive Programming (RxJS):** Deeply integrated for managing asynchronous asynchronous streams, handling HTTP requests, and maintaining state.
*   **Intercepting HTTP Requests:** Custom Angular interceptors are used to automatically attach JWT Bearer tokens to outbound requests and elegantly handle global API errors.
*   **Static Routing for Chatbots:** Employs patterned responses allowing quick client-side navigation paths generated by a real-time chatbot.

## 📦 Key Libraries & Integrations

*   **@microsoft/signalr:** Powers the real-time WebSocket connection to the backend, enabling live customer-support chat.
*   **@abacritt/angularx-social-login:** Allows users to smoothly sign in using their Google accounts.
*   **jwt-decode:** Client-side parsing of secure JWT tokens to extract roles and user profiles.
*   **@ctrl/ngx-emoji-mart:** Implements an interactive emoji picker within the chat interface.

## 🏃‍♂️ Running the Project

To run the application locally in development mode:

```bash
npm install
npm run start
```

The application will launch with Hot Module Replacement (HMR) enabled. Navigate to `http://localhost:4200/`.
